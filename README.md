### npm install - ELIFECYCLE 에러가 나면?

- 그럴 때는 cache를 강제 clean 하고
- node_modules 디렉터리 삭제, 
- package-lock.json을 삭제한 후 다시 npm install을 하면 됩니다.

- $ npm cache clean --force
- $ rm -rf node_modules package-lock.json
- $ npm install
